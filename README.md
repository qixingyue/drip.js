## INSTALL

`
tar zxvf see-3.1.1424.tar.gz
unzip boehm-gc.zip
cd boehm-gc
./configure --prefix=/your/boehm-gc 
make
make install
cd ..
cd see-3.1.1424
./configure --prefix=/your/see --with-boehm-gc=/your/boehm-gc/
make 
make install
`
