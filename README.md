## INSTALL

```
gcpath=/your/boehm-gc
seepath=/your/seepath

tar zxvf see-3.1.1424.tar.gz
unzip boehm-gc.zip
cd boehm-gc
./configure --prefix=$gcpath
make
make install
cd ..
cd see-3.1.1424
./configure --prefix=$seepath --with-boehm-gc=$gcpath
make 
make install
```
